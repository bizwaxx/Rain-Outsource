Rainout Source

Creator: JEEZ Labs

Open agent-readable source for rainout status and game-time play probability for outdoor events. The Austin pilot now supports Krieg Field and Havins Softball Complex.

Live agent endpoints

Service discovery: https://rainout-agent-source.vercel.app

OpenAPI contract: https://rainout-agent-source.vercel.app/openapi.yaml

Supported fields: https://rainout-agent-source.vercel.app/v1/fields

Krieg status example: live status query

Pilot fields

Krieg Field Softball Complex: 515 S Pleasant Valley Rd.

Havins Softball Complex: 12138 N Lamar Blvd.

Rainout phone: 512-978-2680

Official source page: Austin Parks and Recreation Athletics

Weather source: National Weather Service API

Agent rule: Official rainout status wins. Rainout Source checks the Austin Athletics source page, but stays conservative: if no clear cancelled, delayed, or closed signal appears, official status remains unknown and agents should tell the user to call the rainout line before leaving.

Outside-agent test prompt

You are checking an outdoor game status. Use the public Rainout Source API.

1. Open: https://rainout-agent-source.vercel.app
2. Read: https://rainout-agent-source.vercel.app/openapi.yaml
3. Discover supported fields: https://rainout-agent-source.vercel.app/v1/fields
4. Query Krieg Field for: 2026-06-06T20:20:00-05:00
5. Answer in plain English. If official status is unknown after the official source check, do not guess. Tell the user to call the rainout line before leaving.
6. Use the word "rain," not "rainfall."

Example safe answer

Official rainout status is unknown for Krieg Field Softball Complex. At game time, rain chance is 16% with storms possible. Estimated play probability is 50%. Call the rainout line, 512-978-2680, before leaving.
